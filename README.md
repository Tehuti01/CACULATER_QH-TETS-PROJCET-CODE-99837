# Simple JS Calculator

A clean, functional calculator built with fundamental web technologies: HTML, CSS, and JavaScript. This project serves as a practical exercise in front-end development, focusing on user interface design and handling user interactions to perform basic arithmetic operations.

## Features

*   Addition, subtraction, multiplication, and division.
*   Clear (C) and All Clear (AC) functionality.
*   Responsive design that works on different screen sizes.
*   A clean and intuitive user interface.

## Technologies Used

*   **HTML5:** For the structure of the calculator.
*   **CSS3:** For styling and layout.
*   **JavaScript (ES6):** For the application logic and interactivity.

## Project History

This project started as a personal challenge to solidify my understanding of core web development concepts. The initial goal was simple: build a working calculator.

### The Good

The process of creating the user interface with HTML and CSS was straightforward. I enjoyed experimenting with different layouts and color schemes to create a visually appealing and user-friendly design. Implementing the basic digit and operator functionality in JavaScript was also a rewarding experience, as it brought the calculator to life.

### The Hard

The main challenge was handling the logic for the order of operations and managing the state of the calculator. For instance, dealing with sequences of operations, decimal points, and error states (like division by zero) required careful planning and debugging. Initially, my code was a bit messy, with many `if/else` statements.

### What I Learned

This project was a great learning experience. I refactored my JavaScript code multiple times to make it more modular and readable, which taught me the importance of clean code. I also learned a lot about DOM manipulation and event handling in JavaScript. Debugging the calculator's logic improved my problem-solving skills significantly.
